## Number guessing game
A simple Python number guessing game where the player tries to guess a randomly generated number between 0 and 100.
# How to play

1.Computer generate randome number from 0 to 100
2.User try to guess the number
3.For every guess, program updates user:
     - If user guess is too higher
     - If user guess is too low
     - If user guess is correct
4.Program tracks attempt
